# Guessing-Game
This is a simple interactive Number Guessing Game developed using JavaScript, HTML, CSS, and Bootstrap. The application generates a random number between 1 to 100 and allows the user to guess the number with instant feedback until the correct answer is found.

A beginner-friendly JavaScript project where the user must guess a randomly generated number between 1 and 100. The game provides hints such as Too High or Too Low to help the player reach the correct answer.
